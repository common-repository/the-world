== Description ==

The World is a widget plugin to map where you have been around the world so far.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Move TheWorldWidget to you sidebar.
or
3. Place Shortcode [theworld] anywhere in your posts or pages.
